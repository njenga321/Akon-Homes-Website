Build a premium, modern, highly polished real estate developer website called “Akon Homes”.

This is a portfolio-grade frontend project meant to simulate a real-world commercial website for a luxury real estate development company.

TECH STACK:

* Next.js 15 App Router
* TypeScript
* Tailwind CSS
* Framer Motion
* Lucide React icons
* Responsive-first architecture
* Clean reusable component system
* Modern folder structure
* SEO-ready metadata structure
* Smooth page transitions
* Mobile-first responsiveness

OBJECTIVE:
The website must feel:

* luxurious,
* trustworthy,
* modern,
* cinematic,
* corporate,
* conversion-focused.

The site should look comparable to premium real estate brands globally.

DESIGN DIRECTION:
Minimal luxury aesthetic.

Avoid:

* cheap gradients,
* oversaturated colors,
* generic templates,
* excessive animation,
* clutter.

Use:

* restrained motion,
* large whitespace,
* strong typography hierarchy,
* cinematic layouts,
* premium image presentation,
* elegant transitions.

ANIMATION STYLE:

Animations must be:

* subtle,
* smooth,
* premium,

Use:

* staggered grids,

* bouncing,

* flashy effects.
